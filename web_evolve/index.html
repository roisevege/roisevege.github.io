Introduction

This page shows a case study of design evolvement of SLAC Lab webpage from 2000 to 2010. We want to show how the SLAC Lab webpage design evolves from a matimatical aspect.

The original webpages are from Internet Archive. The original dataset is gotten by visual matrix analyzor, which process the pixel information of webpage scrresnhots. You may see the first 10 data in the dataset below.

The last attribute Visual Complexity is an integrated index considering the structure of webpage, image size, color schema and text amount.

From the sample dataset, we can see there are several missing data in the original dataset. The reason for the missing data is that I remove the broken website when fetching the webpage screenshots.
